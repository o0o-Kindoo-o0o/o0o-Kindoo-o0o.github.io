# Kindoo Documentation

Chào mừng bạn đến với hệ thống tài liệu **Kindoo** — nền tảng Odoo ERP được tùy chỉnh và phát triển bởi **Abenla Ltd**.

## Cấu trúc tài liệu

Tài liệu được chia thành **3 phần chính**:

### 📘 Odoo Modules

Tài liệu chính thức về các module chuẩn của Odoo (CRM, Sales, Marketing,...). Phần này giúp người dùng hiểu cách vận hành các tính năng gốc của Odoo ERP.

### 📗 Kindoo User Document Station

Tài liệu hướng dẫn sử dụng dành cho **người dùng cuối** trên hệ thống Kindoo. Bao gồm các hướng dẫn cấu hình và vận hành các module đã được tùy chỉnh riêng cho doanh nghiệp.

### 📙 Kindoo Modules Describe

Tài liệu kỹ thuật mô tả thiết kế và chức năng các **module tự phát triển** trong hệ thống Kindoo ERP, dành cho đội ngũ phát triển và quản trị hệ thống.
